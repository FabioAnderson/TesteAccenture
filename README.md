# Sobre:

Trata-se de uma automação de front-end escrita em ruby, visando validar as funcionalidades da ferramenta de cotação de seguros da tricentis.

Principais pacotes utilizados:

Capybara

Cucumber

Selenium Webdriver

SitePrism

RSpec
